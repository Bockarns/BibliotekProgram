# Individuellet Projekt 
Detta program är ett biblioteksprogram med följande funktioner<br>
*Struktur*<br>
-C#<br>
-SQL<br>
-EFcore<br>

*Funktioner*<br>
-Skapa användare/admin<br>
-Logga in användare/admin<br>
<br>
*Som Admin*<br>
-Lägg till, ta bort eller redigera bok i databas<br>
-Lista samtliga, utlånade eller försenade böcker<br>
-Avaktivera/aktivera användare<br>
-Redigera eget konto<br>
-Avaktivera eller ta bort eget konto<br>
<br>
*Som Användare*<br>
-Sök efter böcker på olika sätt (Författare, titel, ISBN eller nyckelord)<br>
-Låna och lämna tillbaka böcker<br>
-Lista alla, dina eller endast tillgängliga böcker<br>
-Redigera, avaktivera eller ta bort konto<br>
